# kwcmd ———— 一款在windows环境下快速帮助你code的命令行程序

- 版本号：v0.1.4

- 作者：keywet06@wekw

**注意：本程序中 ‘-’ 不与 ‘/’ 、‘\\’ 混用。**

- `$ help` ———— 帮助信息

- `$ clear` ———— 清除本目录下所有多余文件

- `$ run <C++源文件名>` ———— 编译并运行一份C++源代码，其中 `<C++源文件名>` 不包括 “.cpp”

- `$ compile <C++源文件名>` ———— 编译一份C++源代码，其中 `<C++源文件名>` 不包括 “.cpp”

**注意：下载文件后最好将文件名改为 `$.exe` 以符合编译习惯**
